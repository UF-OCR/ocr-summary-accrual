Summary Accrual Tool


Page Overview

The Summary Accrual Import tool allows your to update you summary accrual details into OnCore.

You can export the accrual template below - This template provides the accrual sheet which allows you to prepare your accrual data. The spreadsheet makes it easier to prepare your data for import. You can make changes directly in the spreadsheet and then upload those changes into OnCore via Summary Accrual Import Tool

Preparing for Import

Summary Accrual Import Tool accepts raw accrual data and translates it into the summary accrual record. ** more info on the way

Using the accrual's template spreadsheet

Download the accrual template spreadsheet and copy paste your accrual data into the accruals tab. Do not edit the column name.

Spreadsheet Columns
  • On Study Date*(required): The date of enrollment of the subject. Accepted format MM/DD/YYYY. Must fall between Open to Accrual period for the Institution
  • Institution(optional): Enrolling Institution of the subject. Must match institutions listed in PC Console > Institution. Defaults to University of Florida if left blank.
  • Gender(optional): Description of gender. If the Gender values are different from the OnCore Values (Male, Female and Unknown) then use the Gender tab to provide a mapping scheme.
  • Date of Birth(optional): Participant’s date of birth. Accepted in format (MM/DD/YYYY).
  • Age at Enrollment(optional): Participant age in years at time of enrollment. Only needed when Date of Birth is not available. Accepts numeric value.
  • Ethnicity(optional): Description of participant’s ethnicity. If the Ethnicity values are different from the OnCore Values (Hispanic or Latino, Non-Hispanic, Subject Refused, Unknown) then use the Ethnicity tab to provide a mapping scheme.
  • Race(optional): Description of participant’s Race. If the Race values are different from the OnCore Values (White, Black or African American, Unknown, Native Hawaiian or Other Pacific Islander, Asian, American Indian or Alaska Native, Subject Refused, Unknown) then use the Race tab to provide a mapping scheme.
  • Disease Site (optional, for Oncology only): Description of participant’s disease site. Should match the Data Table 3 Disease Sites as listed in OnCore. Alternately, use the Disease Site tab to provide a mapping scheme.
  • First Name(optional): First Name of the recruited staff.
  • Last Name(optional): Last Name of the recruited staff.
  • Email Address(optional): Email address of the recruited staff. Must be an ufl.edu account.
  • Zip Code(optional): Participant Zip Code at time of enrollment.

The Summary Accrual Import tool prepares the summary of the accrual data received. The user can review this summary on the ‘Review’ screen and decide to either proceed or reupload the data.

Upload the data
** add screen shot

On the ‘Review’ screen click on the button ‘Proceed’. The tool converts your data into summary accrual data and sends it to OnCore. The results are displayed on the screen upon completion. The user can verify the summary accrual data in OnCore(PC Console< Accrual).
